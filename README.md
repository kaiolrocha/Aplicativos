Python Projects:
This repository contains small Python projects developed for learning and practicing programming logic and basic software development concepts.

1_calculadora.py:
A standard calculator developed using CustomTkinter, inspired by the Windows calculator interface. The project was created to explore graphical user interface development in Python and basic arithmetic operations. The application is partially functional and still under development, as it was built primarily for study and experimentation purposes.

2_jogo_da_velha.py:
A Tic-Tac-Toe game developed in Python focusing on fundamental programming logic. The project was intentionally created using only conditional structures (if, elif, else), logical operators (and, or, not), and repetition structures (while, for), aiming to reinforce core programming concepts without relying on advanced libraries or complex abstractions.

Purpose:
These projects were developed as part of academic studies and programming practice, with the goal of improving logical reasoning, coding structure, and familiarity with Python development.
